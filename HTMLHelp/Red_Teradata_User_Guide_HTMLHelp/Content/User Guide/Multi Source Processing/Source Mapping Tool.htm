Source Mapping Tool

A Source Mapping tool is available in WhereScape RED which enables you to graphically map columns from one or more source tables to an existing target table in RED.

This tool provides a graphical representation of the mapped columns between the source tables and the target table. The columns from the source table that exists in the target table are automatically mapped. You can edit the mappings by clicking a source column connection point and dragging the line that appears to the target columnâ€™s connection point.
