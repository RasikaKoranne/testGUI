Gather Statistics

Gathering Statistics on a table will enable the underlying database to optimize each query based on the statistics collected about the data that is being accessed.

Users can either chose to Define Statistics or to Refresh Full Statistics.

Gathering statistics can be performed on any table by selecting this option from a table's right click menu, or to automate this process, by adding a statistics task to a job being processed by the scheduler (Stats, Quick Stats, Analyze or Quick Analyze). For details about adding statistics tasks to jobs, refer to Editing Tasks.
