Index Type

This option enables you to set the default type of Primary Index Type for each table type.

Primary Index Type

Set the default primary index type for each table type.

The options are:

  • Non-Unique Primary Index (NUPI)
  • Unique Primary Index (UPI)
  • No Primary Index (NOPI)

These defaults are applied when an index definition is created.

They can be changed by selecting the Storage tab on the Properties window of an index.
