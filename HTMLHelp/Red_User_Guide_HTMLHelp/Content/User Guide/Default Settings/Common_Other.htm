Other

This option enables you to set the Other user preferences.

Options Description
Trace Unix Sessions Select the checkbox to trace all Unix activity undertaken by the WhereScape RED program until it is terminated. The file WslMedTelnet.txt is created in the program directory for WhereScape RED.
Tip
This option is intended for the debugging of specific Unix problems and the setting of this switch would normally be done at the request of WhereScape when attempting to solve a Telnet issue. This setting is only relevant for the PC on which the setting is made (e.g. it is not a global setting for the repository).
