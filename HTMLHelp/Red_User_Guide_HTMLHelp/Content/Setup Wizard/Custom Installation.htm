Custom Installation

Note:
Before you begin the RED Setup Wizard:
  • Stop any WhereScape RED schedulers running on the computer.
  • Close any other applications that are running.

The following steps describe how install the RED metadata repository and other software components to run RED with the functions and features that you configure.

  1. After successfully installing WhereScape RED, open the RED Windows installation directory and then double-click the RedSetupWizard.exe application to launch the Setup Wizard.
  2. Click the Custom installation type to proceed.
     
  3. The Add License Key window is displayed showing your License details, if you already installed RED before.
    If you are installing RED for the first time, then you need to enter your license information. You can paste your license details or you can browse to a directory and select a License file to use.

    Notes:
    • You can copy the license details currently displayed in the right pane by clicking the copy to clipboard button .
    • If the license key fails to validate, contact support@wherescape.com.
  4. Click Next to continue, the End User License Agreement is displayed. Read the terms of the agreement and then select the I agree to the terms of the license check box.
  5. Click Next to continue, the Metadata Repository window is displayed.
    Select if you want to create a new repository or use an existing repository to store the RED metatdata and then click Next to continue.
