RED Setup Wizard Overview

WhereScape RED provides a metadata installer that enables you to setup and configure your metadata repository settings. The RED Setup Wizard guides you through the process of configuring your metadata repository and the other software components required to run WhereScape RED.

 

The RED Setup Wizard guides you through the following installation processes:

  1. Configuring Metadata Location
  2. Configuring Metadata Repository Settings
  3. Enabling Databases for RED
  4. Configuring Database Enablement Pack Components
  5. Configuring Target Database
  6. Adding Data Sources
  7. Configuring Scheduler for RED
  8. Performing Pre-installation Checks
  9. Reviewing Installation Configuration
  10. Installing RED
  11. Troubleshooting Installation Issues (if necessary)

 

The RED Setup Wizard is launched from the RED Windows installation directory and provides the following installation options:

Option Description
Basic (Typical) Enables you to setup your metadata repository and other software components, using basic recommended settings to run RED with standard functions and features.
Advanced (Custom) Enables you to customize the setup of your metadata repository and other software components, using advanced settings to select functions and features you want to run in RED.
Replay Enables you to use a saved file that contains RED Setup Wizard settings from a previous installation, to setup your metadata repository and other software components to run RED. This option automatically populates settings in the Setup Wizard screens, as you step through the installation process.

 

Note that no matter which type of installation you select, you can change the configured settings later in RED. The steps for each of the above installation options, including the pre-requisites for configuring the RED metadata repository and other associated settings are described in the succeeding sections.
