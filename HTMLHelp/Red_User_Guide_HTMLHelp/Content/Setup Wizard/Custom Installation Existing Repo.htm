Custom Installation - Existing Repository

  1. Selecting the Use existing repository option displays the Existing Metadata Database window that enables you to specify a data source to which RED connects and the log in details for the selected source database. Click Validate to continue.
  2. After successful validation, click Next to continue the data source and log in credentials validation.
    3. Note:

    If there is a difference in the metadata repository version you are upgrading, then the RED Metadata Validate application is launched, which guides you through the steps to upgrade your metadata repository to match the RED version you are installing.
    Refer to the RED Metadata Validate application user guide for details.
  3. Once the installation is complete, a message is displayed to confirm that the setup was successful (same as step 11 in New Repository installation).
  4. Click Finish to launch RED and start working with your data warehouse.
