Managing the RED Licenses

The Install WhereScape RED License window enables you to perform the following:

  • Display currently installed license information
  • Install New License
  • Import License File
  • Copy License details
  • Reload License details

 

The following describe the elements and functions of this screen:

Options Description
Show License Information This option displays the decrypted details of the currently installed license which includes the following:
  • Metadata Database(s) - The database(s) that you can setup for the metadata repository or data warehouse, e.g. SQL Server, Oracle, Teradata, etc.
  • Target Database(s) - The database(s) that you can setup for the target database or where the physical data warehouse table objects are created, e.g. SQL Server, Oracle, Custom, etc.
  • Licensed Model Types(s) - The data warehouse model(s) that you can use to design, develop and create your data warehouse, e.g. Star Schema, Data Store, 3NF, Data Vault, Custom, etc.
  • Licensed Users - The number of users included in your license.
  • Licensed Usage Type - The type of users included in your license.
  • Restriction(s) - The restriction(s) that are applied to your license.
  • Expiry - The expiration date of your license.
  • Custom Target Database - The custom database(s) that you can setup for the target database or where the physical data warehouse table objects are created, e.g. Snowflake, Amazon Redshift, Microsoft Azure, etc.
Show Raw License Key This option displays the raw, encrypted details of the currently installed license which includes the following:
  • Licensed To - The name of the organization licensed to use WhereScape RED.
  • License Key - The encrypted license key data.
Copy button This button enables you to copy the details of the license currently displayed in the right pane to the Windows clipboard.
Reload button This button enables you to reload the details of the license currently displayed in the right pane.
Install New License This button opens the Install License Key window which enables you to enter or paste license information provided by your WhereScaperepresentative.
Import License File This button opens the Windows File Explorer which enables you to navigate to a directory and select your License file that contains your license information.
