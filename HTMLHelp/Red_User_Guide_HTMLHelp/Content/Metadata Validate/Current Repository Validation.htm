Current Repository Validation

Notes:
Ensure the scheduler is not running and no users are accessing the metadata before doing an upgrade.

Use this option if you want to validate a RED metadata repository that you are currently logged.

  1. Log into RED and click the Validate menu to select the Validate Metadata Repository option.

  2. Click the Current metadata repository option to proceed.

  3. The Metadata Validate tool connects to the current metadata repository and checks if the latest version of the metadata schema has been applied and if there are changes to the table objects in the repository to determine the next processing actions to perform.

The succeeding steps for each scenario is described in the following sub sections:
