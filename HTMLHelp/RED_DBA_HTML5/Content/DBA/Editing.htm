Editing

The edit pane of the Sql Admin window allows for the entering and editing of sql command scripts. A command script can consist of one or more sql statements. A statement is deemed to be everything up to a terminating semi-colon. All statements in the script are executed sequentially when the Execute button is clicked. The normal Windows select, cut, paste, find and replace operations may be performed in this pane. Each line in the edit pane is terminated by a carriage return/line feed combination.

Comments

Comments may occur within the sql statement. When encountered they are removed prior to passing the statement to the database. Comments are identified by two concurrent dash characters (i.e. --). When encountered all remaining text on the line is ignored.

Including other script files

Additional script files can be included at execution time by using the $sql_include prefix. When encountered at execution time the contents of these scripts are read and executed as if they had been entered in the edit pane.
