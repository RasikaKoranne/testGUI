Teradata Installation and Configuration

Introduction

The steps provided here are an indication of the normal process required to install and configure WhereScape RED. In some environments additional processes will be required.

  1. Install Teradata Tools and Utilities 13.10 (or greater) or on a suitable computer. These tools and utility components are required:
    • CLI
    • Teradata ODBC Driver
    • BTEQ
    • Fast Load / Multiload or TPT
    • Fast Export
  1. Install the WhereScape RED software onto your PC. Refer to Installing RED for Teradata for details.
  1. Add your license key. Refer to Adding your License for details.
  1. Check your Windows version. Refer to Checking your Windows Version for details.
  2. Check your ODBC Drivers. Refer to Checking your ODBC Drivers for details.
  1. Designate a database or user that is to be used as the meta data database. For testing or evaluation purposes this database or user may be a Teradata Express database server. WhereScape RED requires the Teradata database version 15.10 or greater as the data warehouse database server. Record the database and server names for this database. Create a database user to hold the metadata tables.
  1. Allocate the privileges required by WhereScape RED. Refer to Creating a Teradata Environment for details.
  1. Add appropriate entries to the hosts file on the computer for the server.
  1. Add a definition of an ODBC data source to connect to the data warehouse database. Refer to Configuring a Teradata Metadata Connection (DSN) for details.
  1. Create the WhereScape metadata repository under the data warehouse user that you created. Refer to Creating a Metadata Repository for Teradata for details.
  1. Install a Windows Scheduler. Refer to Installing a Windows Scheduler for Teradata for details
  1. Install a Unix Scheduler. Refer to Installing a Unix Scheduler for Teradata for details.
  1. Install a Teradata Unix Loader. Refer to Installing a Teradata Unix Loader for details.
  1. If required, load the WhereScape tutorial tables into a separate database. It is highly recommended that the tutorials be undertaken. Refer to Creating Tutorial Data for details.

Problem Solving

If problems occur in your install that cannot be resolved, proceed as follows when running WhereScape's Setup Administrator utility:

  1. Create a log file by selecting Tools > Start Logging from the menu bar. Enter a file name.
  1. Step through at least:
    • Computer Setup > Check Windows version.
    • Computer Setup > Check ODBC driver.
    • Computer Setup > Validate/Add Odbc source.
    • Create/Validate Meta Repository > Validate meta repository.
  1. Select Tools > Stop Logging to close off the log.
  1. Edit the log and remove any passwords that appear.
  1. Send your log file to support@wherescape.com.
